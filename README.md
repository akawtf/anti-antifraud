# 反反诈中心

## 这是什么？
这是个**假的**“国家反诈中心”，可以认为是“国家反诈中心”的疫苗

## 有什么用？
有人问你“装了没？”，你可以理直气壮地说“装了！”

## 打开以后什么效果？
界面会卡死，这时候顺便可以吐槽他们的APP写得真烂

## 为什么要做这个？
防止手机被强制安装垃圾软件
